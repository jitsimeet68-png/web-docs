# Curated Functions Repository

## Summary
- Repository aggregates custom functions vetted by the Open WebUI core team, signaling production-ready quality.
- README positions the collection as easy to integrate and links back to official plugin/function guides for deeper reference.
- Directs users to the broader community marketplace at openwebui.com for additional contributions.

## Usage Guidelines
- Treat listed functions as trusted starting points when seeding an Open WebUI deployment.
- Review linked documentation pages before installing to understand function types and lifecycle management.
- Encourage teams to contribute battle-tested functions back to the repository for central visibility.

## Key Terms
- **Curated Function** – Community contribution reviewed and approved by core maintainers.
- **Function Catalog** – Repository acting as a hub for high-quality extensions.
- **Community Marketplace** – openwebui.com listings for broader function discovery.

## Implementation Notes
- Cross-reference repository updates when maintaining internal allowlists of approved functions.
- Incorporate README links into onboarding materials to connect developers with canonical docs.
- Monitor repository issues/releases for updates affecting dependent deployments.
